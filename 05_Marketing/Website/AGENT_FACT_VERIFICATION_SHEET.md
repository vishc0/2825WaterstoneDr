# Agent fact and verification sheet (sample)

Use internally with the listing representative. This is not a public MLS replacement.

## Source labels

- **Listing fact:** current FMLS/GAMLS source record.
- **Owner-described:** seller experience or description; attribute it.
- **Verify:** obtain documents, measurements, HOA confirmation or realtor approval.

## Priority checks

- [ ] FMLS and GAMLS both show 6 bedrooms / 5 full baths.
- [ ] Second-floor media/flex room and basement game/flex room are not counted as official bedrooms unless formally verified.
- [ ] Attached two-car garage matches the current realtor listing.
- [ ] 6,377 sq ft source and 4,322/2,055 breakdown supported by appraisal or professional measurement.
- [ ] Full LVP, bedroom carpet replacement and redone stairs confirmed in remarks/structured fields.
- [ ] Primary bath quartz, cabinetry, shower setup and upgraded toilets confirmed.
- [ ] HOA, schools, routes, Dave’s Creek adjacency and nearby-distance claims verified.
- [ ] Event, parking, occupancy and fire requirements reviewed before discussing large gatherings.
