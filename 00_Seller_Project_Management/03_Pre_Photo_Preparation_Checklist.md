# Pre-Photo Preparation Checklist

Purpose: Make the home photograph as bright, spacious, current, and premium.

## 7 Days Before Photos

- [ ] Confirm photographer date.
- [ ] Confirm stager date.
- [ ] Confirm cleaner date.
- [ ] Confirm landscaping/mulch/lawn timing.
- [ ] Confirm any handyman tasks.
- [ ] Confirm twilight/drone/sunset photo availability.
- [ ] Confirm whether exterior uplighting or temporary lighting will be ready.
- [ ] Remove excess furniture if rooms feel crowded.
- [ ] Start packing visible personal items.
- [ ] Confirm final kitchen hardware installation.
- [ ] Confirm final fixture/fan decisions.

## 3 Days Before Photos

- [ ] Deep clean kitchen.
- [ ] Deep clean bathrooms.
- [ ] Clean windows and glass doors.
- [ ] Clean deck surfaces.
- [ ] Sweep garage.
- [ ] Clear basement storage spillover.
- [ ] Touch up visible paint marks.
- [ ] Replace burned-out bulbs.
- [ ] Set visible lighting to warm/tunable white, not colored RGB.
- [ ] Hide fireplace/TV wires and visible technology cords.
- [ ] Test reed diffusers only after deep cleaning.
- [ ] Remove refrigerator magnets, papers, and personal items.

## 1 Day Before Photos

- [ ] Hide trash cans.
- [ ] Hide cleaning supplies.
- [ ] Hide laundry baskets.
- [ ] Hide pet items.
- [ ] Hide toiletries.
- [ ] Hide cords.
- [ ] Clear counters.
- [ ] Make all beds.
- [ ] Fluff pillows.
- [ ] Put fresh towels in bathrooms.
- [ ] Stage decks.
- [ ] Move cars away from driveway and front curb.

## Photo Morning

- [ ] Turn on all lights.
- [ ] Open blinds.
- [ ] Close toilet lids.
- [ ] Remove floor mats if they make rooms look smaller.
- [ ] Remove dish towels unless stager wants them.
- [ ] Remove visible remotes.
- [ ] Wipe counters.
- [ ] Wipe mirrors.
- [ ] Place subtle reed diffusers only in foyer, living area, and primary bath if scent strength is mild.
- [ ] Keep kitchen fragrance-free.
- [ ] Check front door and porch.
- [ ] Check decks.
- [ ] Check basement lighting.
- [ ] Keep people and pets out of photographer path.

## Must-Show Photo Moments

- [ ] Front exterior
- [ ] Two-story foyer
- [ ] Office with glass doors
- [ ] Open kitchen/living connection
- [ ] Fireplace/living focal point
- [ ] Formal dining room
- [ ] Primary suite
- [ ] Primary bath
- [ ] Basement in-law suite kitchen
- [ ] Media room
- [ ] Game room
- [ ] Double decks
- [ ] Creek-lot lifestyle
- [ ] Three-car garage
