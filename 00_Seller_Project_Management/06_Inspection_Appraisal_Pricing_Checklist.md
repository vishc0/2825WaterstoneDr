# Inspection, Appraisal, and Pricing Checklist

Purpose: Reduce buyer uncertainty, protect negotiation leverage, and keep pricing decisions grounded.

## Appraisal Reality Check

- [ ] Save appraisal PDF in `01_Property_Documents`.
- [ ] Save appraiser remarks in `01_Property_Documents`.
- [ ] Note current appraiser opinion of value: `$1,100,000`.
- [ ] Note appraisal constraint: only one Waterstone Falls sale above `$1,000,000` was identified in the appraiser remarks.
- [ ] Note premium competition risk: newer/gated/currently listed homes near `$1.23M+` can make a higher list price harder to defend.
- [ ] Ask realtor which comps support the final list price.
- [ ] Ask realtor whether the target strategy is appraisal-safe pricing, aspirational pricing, or staged high with planned adjustment.

## CMA and Market Review

- [ ] Perform detailed CMA using recent 6-12 month sold comps.
- [ ] Compare Waterstone Falls sales.
- [ ] Compare Windermere and nearby executive-home sales.
- [ ] Review active listings in the `$1.0M-$1.3M` range.
- [ ] Review pending listings if realtor can access them.
- [ ] Compare appraisal value to renovated/staged condition.
- [ ] Decide final pricing only after staging and photography quality are known.

## Pre-Listing Inspection Package

- [ ] Decide whether to order a full pre-listing home inspection.
- [ ] Obtain HVAC inspection or certification.
- [ ] Obtain roof inspection or certification.
- [ ] Obtain termite inspection or bond documentation.
- [ ] Consider sewer scope only if realtor believes buyers in this market expect it.
- [ ] Save all reports in `02_Inspection_Package`.

## Known Buyer Objection Controls

- [ ] Confirm HVAC age, service history, and remaining-life comment if possible.
- [ ] Confirm roof age, repair history, and remaining-life comment if possible.
- [ ] Verify basement permit/status documentation if available.
- [ ] Document tankless water heater.
- [ ] Document water management and drainage improvements.
- [ ] Document deck repairs/improvements.
- [ ] Prepare a concise explanation for original systems that remain.

## Pricing Decision Prep

- [ ] Discuss `$1,199,000` as a psychological threshold option if targeting around `$1.2M`.
- [ ] Discuss whether `$1,249,900` is supportable after staging, photography, and CMA.
- [ ] Decide private minimum acceptable price.
- [ ] Decide first price-adjustment trigger: number of showings, feedback pattern, or days on market.
- [ ] Prepare appraisal-support packet before offers arrive.
