# Staging and Showing Checklist

Purpose: Turn the home into a memorable executive, creekside, multi-generational property rather than a standard suburban listing.

## Staging Strategy

- [ ] Use Modern Warm Transitional Luxury as the design direction.
- [ ] Avoid farmhouse, heavy traditional, overly gray, and overly contemporary styling.
- [ ] Prioritize staging over broad additional renovation unless a defect will hurt photos.
- [ ] Decide full vs partial staging after quotes.
- [ ] Prioritize rooms by buyer impact.

## Highest-Impact Staging Zones

- [ ] Two-story foyer: clean arrival, mirror/art/greenery if appropriate.
- [ ] Living room/fireplace: make this a hero photo; hide TV wires.
- [ ] Kitchen: clear counters, highlight white cabinets, backsplash, island, and hardware.
- [ ] Dining room: confirm chandelier and stage for scale.
- [ ] Office with glass doors: stage as serious executive workspace.
- [ ] Primary suite: hotel bedding, balanced nightstands, spa bath styling.
- [ ] Basement in-law suite: make it feel complete, independent, and dignified.
- [ ] Media room: make entertainment use obvious.
- [ ] Game room: make flexible family/recreation use obvious.
- [ ] Upper deck: stage heavily as outdoor living.
- [ ] Second deck: stage as coffee, dining, or quiet retreat.
- [ ] Garage: clear enough to show 3-car functionality.

## Scent and Sensory Plan

- [ ] Deep clean before adding scent.
- [ ] Use subtle reed diffusers, not candles.
- [ ] Preferred scent profile: white tea, bergamot, eucalyptus, linen, or cedar.
- [ ] Place only in foyer, living area, primary bath, and possibly basement if needed.
- [ ] Keep kitchen fragrance-free.
- [ ] Test scent strength before showings; remove if noticeable or artificial.

## Daily Showing Reset

- [ ] Counters clear.
- [ ] Beds made.
- [ ] Toilets closed.
- [ ] Towels fresh.
- [ ] Lights on.
- [ ] Blinds open.
- [ ] Basement lights on.
- [ ] Decks swept.
- [ ] Trash hidden.
- [ ] Pet items hidden.
- [ ] Laundry hidden.
- [ ] Cords hidden.
- [ ] Valuables, medicine, documents, and electronics secured.

## Open House / Agent Preview

- [ ] Ask realtor whether agent preview is worthwhile.
- [ ] Prepare buyer-agent packet.
- [ ] Prepare QR code to digital due-diligence room.
- [ ] Keep printed upgrade sheet available.
- [ ] Keep buyer FAQ available.
- [ ] Keep inspection summary available if reports are clean and useful.
