# Master Sequential Seller Checklist

Property: 2825 Waterstone Drive, Cumming, Georgia

Use this as the main execution checklist. Work from Phase 1 downward.

## Phase 1: Organize the Project

- [ ] Confirm target listing date or desired photography date.
- [ ] Confirm realtor, photographer, stager, cleaner, landscaper, handyman, and any remaining contractors.
- [ ] Save all project files into the numbered folders.
- [ ] Add exported ChatGPT project conversations into `chats` if available.
- [ ] Review `AiDirection.md` so future AI work stays aligned.

## Phase 2: Collect Seller Documents

- [ ] Collect renovation receipts.
- [ ] Collect appraiser remarks and appraisal PDF in one easy-to-find location.
- [ ] Collect appliance model numbers, manuals, and warranty details.
- [ ] Collect HOA documents.
- [ ] Collect utility bills or average monthly costs.
- [ ] Collect roof repair or inspection records.
- [ ] Collect HVAC service records.
- [ ] Collect termite records.
- [ ] Verify basement permit/status documentation, if available.
- [ ] Collect water management, drainage, and landscaping improvement records.
- [ ] Collect smart-home device details.
- [ ] Collect floor dimensions if available.

Save documents in:

- `01_Property_Documents`
- `02_Inspection_Package`

## Phase 3: Decide Final Pre-Listing Improvements

- [ ] Walk the home room by room.
- [ ] Identify visible defects that could hurt photos.
- [ ] Identify small repairs that buyers may overreact to.
- [ ] Separate tasks into HIGH ROI, MEDIUM ROI, and LOW ROI.
- [ ] Complete only the highest-impact items first.
- [ ] Confirm kitchen hardware finish and consistency.
- [ ] Confirm dining chandelier / key light fixture plan.
- [ ] Confirm whether builder-grade ceiling fans should remain or be replaced.
- [ ] Confirm smart recessed lighting plan: use tunable white as the main strategy; avoid RGB as the primary buyer-facing feature.
- [ ] Decide whether under-cabinet lighting is worth doing before photos.
- [ ] Decide whether any closet upgrade is still worth the time/cost.
- [ ] Decide whether any kitchen countertop change is still worth the time/cost.
- [ ] Confirm deck boards, railings, stairs, and stain/paint readiness.
- [ ] Hide visible TV/fireplace wiring before photography.

Priority order:

1. Front exterior and entry
2. Kitchen
3. Main living area
4. Primary suite
5. Office
6. Basement/in-law suite
7. Decks
8. Garage and storage areas

## Phase 4: Staging Preparation

- [ ] Review `05_Marketing/Staging_Consultant_Brief.md`.
- [ ] Contact staging consultants from `05_Marketing/Staging_Consultant_Shortlist.md`.
- [ ] Contact all top staging options the same day; do not wait for one reply at a time.
- [ ] Book the fastest qualified staging consultation.
- [ ] Declutter before the consultation.
- [ ] Ask stager to prioritize occupied-home styling and photo impact.
- [ ] Confirm if stager can return on photo day.
- [ ] Decide full vs partial staging after quotes.
- [ ] Ask specifically about deck/outdoor staging and basement/in-law suite staging.
- [ ] Ask for portfolio examples relevant to Forsyth County, Alpharetta, Johns Creek, or North Atlanta executive homes.

## Phase 5: Pre-Photography Preparation

- [ ] Deep clean the entire home.
- [ ] Clean windows and glass doors.
- [ ] Touch up paint where visible.
- [ ] Replace burned-out bulbs.
- [ ] Use consistent warm-white bulbs where possible.
- [ ] Remove personal photos and private documents.
- [ ] Hide cords, trash cans, cleaning supplies, pet items, and excess small appliances.
- [ ] Prepare decks with clean furniture, cushions, and planters.
- [ ] Clear garage enough to show 3-car functionality.

## Phase 6: Photographer Coordination

- [ ] Confirm photo date and arrival time.
- [ ] Confirm shot list with realtor and photographer.
- [ ] Confirm whether twilight, drone, and creek/deck shots are included.
- [ ] Confirm whether sunset deck photos are possible.
- [ ] Confirm whether floor plans or CubiCasa-style plans will be created.
- [ ] Confirm photo-day cleaning and staging touch-up timing.
- [ ] Open blinds and turn on lights before photographer arrives.
- [ ] Keep cars out of driveway for exterior shots.

## Phase 7: Marketing Package

- [ ] Build property story.
- [ ] Build buyer FAQ.
- [ ] Build upgrade portfolio.
- [ ] Build buyer confidence binder / digital due-diligence room.
- [ ] Build appraisal-support packet: appraisal, comps, upgrade summary, inspections, and feature list.
- [ ] Build smart-home guide.
- [ ] Build community guide.
- [ ] Prepare brochure copy.
- [ ] Prepare Zillow/listing copy.
- [ ] Prepare social media captions.
- [ ] Prepare realtor talking points.
- [ ] Select property website platform.
- [ ] Check domain options such as `2825Waterstone.com` or `Live2825Waterstone.com`.
- [ ] Create QR code plan for binder, brochure, website, and open house.
- [ ] Plan video assets after photography: teaser, lifestyle walkthrough, and full feature/property story video.

Save materials in:

- `05_Marketing`
- `06_Buyer_Questions`

## Phase 8: Showing Readiness

- [ ] Create a daily showing reset checklist.
- [ ] Keep counters clear.
- [ ] Keep beds made.
- [ ] Keep bathrooms photo-ready.
- [ ] Keep decks clean.
- [ ] Keep basement bright and organized.
- [ ] Keep temperature comfortable.
- [ ] Prepare a secure place for valuables, medicine, documents, and electronics.

## Phase 9: Offer and Negotiation Readiness

- [ ] Prepare list of preferred terms.
- [ ] Decide minimum acceptable price privately.
- [ ] Decide preferred closing date.
- [ ] Decide concession strategy.
- [ ] Prepare inspection response strategy.
- [ ] Prepare appraisal-risk backup documentation.
- [ ] Discuss whether a buyer home warranty should be offered strategically.
- [ ] Discuss buyer-agent packet and agent preview strategy with realtor.
- [ ] Save all offer comparisons in `08_Negotiation`.

## Phase 10: Weekly Review

- [ ] Review completed tasks.
- [ ] Review open blockers.
- [ ] Review buyer/realtor feedback.
- [ ] Update marketing or staging if feedback repeats.
- [ ] Keep project files updated.
