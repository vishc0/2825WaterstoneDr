# Document Collection Checklist

Purpose: Build buyer confidence, support premium pricing, reduce inspection objections, and help the realtor answer questions quickly.

## Highest Priority

- [ ] Appraisal PDF
- [ ] Appraiser remarks email
- [ ] Any CMA or realtor pricing analysis
- [ ] Recent sold comps from realtor
- [ ] Active competing listings in $1.0M-$1.3M range
- [ ] HOA documents
- [ ] Recent utility costs
- [ ] Renovation receipts
- [ ] Flooring documentation
- [ ] Interior paint details
- [ ] Kitchen cabinet refinishing details
- [ ] Backsplash details
- [ ] Fireplace renovation details
- [ ] Bathroom quartz upgrade details
- [ ] New toilet receipts/details
- [ ] Tankless water heater details
- [ ] HVAC service records
- [ ] Roof repair or inspection records
- [ ] Termite records
- [ ] Transferable termite bond details, if available
- [ ] Any pre-listing home inspection report, if ordered
- [ ] Any roof certification, if ordered
- [ ] Any HVAC certification, if ordered

## Property Systems

- [ ] Electrical updates
- [ ] Plumbing updates
- [ ] Water heater documentation
- [ ] HVAC filters and maintenance schedule
- [ ] Remaining HVAC life estimate, if vendor will provide it
- [ ] Remaining roof life estimate, if vendor will provide it
- [ ] Smart-home device list
- [ ] Wired networking details
- [ ] Security/camera/device transfer notes
- [ ] Garage door opener details

## Exterior and Lot

- [ ] Landscaping improvement records
- [ ] Drainage or water management records
- [ ] Deck improvement records
- [ ] Creek/lot maintenance notes
- [ ] HOA exterior approval records, if any

## Appliances

- [ ] Refrigerator model and age
- [ ] Dishwasher model and age
- [ ] Oven/range model and age
- [ ] Microwave model and age
- [ ] Washer/dryer model and age, if included
- [ ] Basement/in-law suite appliance details

## Buyer Binder Candidates

- [ ] Appraisal-support summary
- [ ] Upgrade summary
- [ ] Maintenance summary
- [ ] Utility summary
- [ ] Smart-home guide
- [ ] Floor plan or dimensions
- [ ] Community guide
- [ ] School/location notes
- [ ] Buyer FAQ
- [ ] Inspection summary
- [ ] QR code index for digital due-diligence room

## Filing Rule

Save source documents in `01_Property_Documents`.

Save inspection, roof, HVAC, termite, and repair documentation in `02_Inspection_Package`.
