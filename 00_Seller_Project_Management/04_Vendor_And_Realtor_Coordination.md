# Vendor and Realtor Coordination

Purpose: Keep all outside parties aligned and reduce missed details before listing.

## Contacts to Confirm

- [ ] Realtor
- [ ] Photographer
- [ ] Stager
- [ ] Cleaner
- [ ] Landscaper
- [ ] Handyman
- [ ] HVAC vendor, if needed
- [ ] Roof vendor, if needed
- [ ] Termite/pest vendor, if needed

## Realtor Questions

- [ ] What is the target listing date?
- [ ] What is the target photography date?
- [ ] What list price range are we preparing for?
- [ ] How should we reconcile the $1,100,000 appraisal with desired listing strategy?
- [ ] Which comps will buyers and appraisers likely use?
- [ ] Which comps should be included in the appraisal-support packet?
- [ ] Are there current active competitors near $1.0M-$1.3M we should monitor?
- [ ] Which features should be emphasized most?
- [ ] What objections do you expect from buyers?
- [ ] Should we prepare a printed property binder?
- [ ] Should we prepare a digital property website?
- [ ] Do you want upgrade receipts before listing?
- [ ] What should be disclosed, repaired, or documented before showings?
- [ ] Should we order roof, HVAC, termite, or full pre-listing inspection documentation before launch?
- [ ] Should we offer a buyer home warranty?
- [ ] Should we host an agent preview event?

## Photographer Questions

- [ ] Are drone photos included?
- [ ] Are twilight photos included?
- [ ] Can we schedule around sunset for rear deck impact?
- [ ] Will the creek/deck setting be captured?
- [ ] Will basement/in-law suite photos be emphasized?
- [ ] Will office/glass-door work-from-home value be emphasized?
- [ ] Will three-car garage functionality be captured?
- [ ] How long will photography take?
- [ ] What must be ready before arrival?

## Stager Questions

- [ ] Can you consult quickly?
- [ ] Can you support occupied-home staging?
- [ ] Can you provide accessory rental only?
- [ ] Can you style for photography?
- [ ] Can you return on photo day?
- [ ] Which rooms are highest ROI?
- [ ] Can you stage the upper deck as a luxury outdoor living moment?
- [ ] Can you make basement/in-law suite feel complete and dignified?
- [ ] Can you provide bedding, art, pillows, greenery, and outdoor accents?
- [ ] Can you work from photos/video first if in-person scheduling is delayed?

## Cleaner Questions

- [ ] Can you clean before photography?
- [ ] Are windows included?
- [ ] Are glass doors included?
- [ ] Are baseboards included?
- [ ] Are kitchen cabinets wiped externally?
- [ ] Are bathrooms detailed?

## Landscaper Questions

- [ ] Can the lawn be photo-ready before photography?
- [ ] Can mulch/pine straw be refreshed?
- [ ] Can shrubs be shaped?
- [ ] Can walkways and deck areas be blown clean?
- [ ] Can creek/deck view areas be cleaned without over-clearing?

## Coordination Rule

Schedule vendors in this order:

1. Handyman and repairs
2. Landscaping
3. Staging consultation
4. Cleaning
5. Staging touch-up
6. Photography
